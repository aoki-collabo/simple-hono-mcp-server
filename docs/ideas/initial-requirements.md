# 最もシンプルなリモートmcpサーバー

リモートMCPサーバーについて、cloudflareとhonoを用いてデプロイすることの簡単さを伝えたいと思っています。そのために簡単な技術ブログとして記事を書こうと思っています。その題材のサンプルのアプリケーションを作成します。

typescriptを用います。デモのため、npmとnpxを用いる方が話が早いと思いっています。
依存するパッケージは最新のものを使いたいと思っています。

mcpのツールとしては、数値を受け取って素因数分解を行ってデータを返すツールにしたいと思います。かかった時間とかも一緒に返すのも面白いかもです。

サンプルのため、認証などはなしにしたいと思います。
cloudflare側の設定も少なくしたいと思います。

README.mdにデプロイ方法やローカルでのお試し方法を記載する。

## アプリケーション仕様

### 基本情報
- アプリケーション名: simple-mcp-server
- トランスポート: Streamable HTTP

### ツール: 素因数分解 (factorize)

#### 入力
- 正の整数（上限あり、例: 15桁まで）

#### 出力
- 素因数のリスト
- 割り算を行った回数

#### エラー処理
- 親切な日本語メッセージを返す
  - 例: 「正の整数を入力してください」
  - 例: 「入力値が上限を超えています」

### 非機能要件
- 認証: なし
- Cloudflare設定: 最小限
